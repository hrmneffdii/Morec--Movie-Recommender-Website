# User API Address

## Method Get 

Menampilkan semua data dari database.

Endpoint url:'/api/users'

Request : 

```json
[
    {
        "nama" : "nama1",
        "email": "email1@gmail.com",
        "password": "password"
    }
    ...
    {
        "nama": "nama-n",
        "email":"email.n@gmail.com",
        "password" : "password-n"
    }
]
```
